# Netflix Clone App

A Netflix clone application built using web technologies including React, React-Redux, TailwindCSS, MongoDB, Node.js, and Express. This project showcases a full-stack web application with a focus on front-end design and back-end functionality.

## Features

- **User Authentication**: Sign up, login, and secure access to user accounts.
- **Browse Movies and TV Shows**: Browse through a wide range of movies and TV shows.
- **Search Functionality**: Search for your favorite movies and TV shows.
- **Responsive Design**: Fully responsive design using TailwindCSS.
- **User Profiles**: Manage user profiles and personalized watch lists.
- **Video Streaming**: Stream video content seamlessly.

## Technologies Used

### Front-End

- **React**: A JavaScript library for building user interfaces.
- **React-Redux**: A predictable state container for JavaScript apps.
- **TailwindCSS**: A utility-first CSS framework for rapid UI development.

### Back-End

- **Node.js**: A JavaScript runtime built on Chrome's V8 JavaScript engine.
- **Express**: A minimal and flexible Node.js web application framework.
- **MongoDB**: A NoSQL database for storing user data and media information.
